## 📦️ 安装

访问 [Releases](https://github.com/jinghuashang/Hydrogen-Music/releases)
页面下载安装包。

## 👷‍♂️ 打包客户端

因为`Snap`打包特殊性只打包了Windows平台和macOs的安装包且并未适配Linux平台。
如有可能，您可以在开发环境中自行适配。

```shell
# 打包
npm run dist
```
## :computer: 配置开发环境

运行本项目

```shell
# 安装依赖
npm install

# 运行Vue服务
npm run dev

# 运行Electron客户端
npm start
```
